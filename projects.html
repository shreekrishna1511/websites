Bottle RC Plane

This project focuses on designing a simple, stable RC trainer plane suitable for beginners using bottles of Coke and Fanta. It uses a NACA 6412 airfoil and basic control surfaces. It's ideal for learning aerodynamics and RC piloting skills at a low cost using recycled materials.
